https://api.json-generator.com/templates/M-rnKhH0Z6uf/data?access_token=edfgbmrjms4yldm77epct3jgyo3q65wxjinddzmc <br>

.NET 6.0<br>
Windows OS<br>
Visual Studio 2022

<h2>Build Instructions</h2>
